## üí° About this Project

Employee Management System (MERN) ‚Äì A full-stack web application to manage employees efficiently. 
Built with MongoDB, Express, React, and Node.js, it supports adding, updating, deleting, and viewing employees.
Features include image upload with preview, single course selection, JWT-based authentication, and real-time notifications with react-hot-toast.
Perfect for learning MERN stack CRUD operations with file handling and protected routes.
This project contains **two main parts**:  
1. **Backend** ‚Äì API built with Node.js, Express, and MongoDB.  
2. **Frontend** ‚Äì React-based UI styled with TailwindCSS.

## üö¶ Running the Project

Follow these steps to run locally.

---

### 1Ô∏è‚É£ Prerequisites

- Node.js installed
- MongoDB Atlas account or a local MongoDB instance
- Git installed

---

### 2Ô∏è‚É£ Clone the Repository

```bash
git clone https://github.com/Deepak-0808/EmployeeManagement.git
cd EmployeeManagement
```

### 3Ô∏è‚É£ Backend Setup
```bash
cd backend
npm install
```

Create .env file in backend folder:
```bash
PORT=5000
MONGO_URI=mongodb+srv://<username>:<password>@cluster.mongodb.net/EmployeeManagement
```

Run Backend Server:
```bash
npm start
```

The backend will be available at:
http://localhost:5000

### 4Ô∏è‚É£ Frontend Setup
```bash
cd ../frontend
npm install
```

Create .env file in frontend folder:
```bash
REACT_APP_BASE_URL=http://localhost:5000
```

Run Frontend Server:
```bash
npm run start
```
The frontend will be available at:
http://localhost:5173
